** Module flags
- +lsp ::
  Enable LSP support for ~yaml-mode~. Requires [[rmcs-module::tools lsp]] and a langserver
  (supports [[https://github.com/redhat-developer/yaml-language-server][yaml-language-server]]).
- +tree-sitter ::
  Leverages tree-sitter for better syntax highlighting and structural text
  editing. Requires [[rmcs-module::tools tree-sitter]].
